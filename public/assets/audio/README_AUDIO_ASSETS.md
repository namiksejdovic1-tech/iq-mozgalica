# Audio Assets Guide

Please upload your generated audio files to the following folders.
Filenames must match exactly for the code to work.

## SFX (`/public/assets/audio/sfx/`)

- `sfx_startup.mp3` (Startup chime)
- `sfx_victory_shimmer.mp3` (Win sound)
- `sfx_soft_thud.mp3` (Error/Retry sound)
- `sfx_precision_click.mp3` (Logic master sound)
- `sfx_air_release.mp3` (Break time sound)
- `click-modern.mp3` (General UI click)
- `success-shimmer.mp3` (General success)
- `error-thud.mp3` (General error)
